# 🍱 Intelligent Food Distribution Management System

A cloud-enabled web application developed to reduce food wastage by connecting food donors with people in need through an intelligent food distribution platform.

---

## 📌 Project Overview

The Intelligent Food Distribution Management System helps donors share surplus food with receivers in real time. The application provides a secure and user-friendly platform with separate dashboards for Admin, Donor, and Receiver.

This project was developed as a Final Year Engineering Project for B.E. Computer Science and Engineering.

---

## ✨ Features

- 🔐 Secure Login & Registration
- 👨‍💼 Admin Dashboard
- 🍱 Donor Dashboard
- 🥣 Receiver Dashboard
- 📦 Food Donation Management
- 📍 Pickup Location Details
- 📩 Food Request Management
- 🔔 Notification System
- ☁️ AWS EC2 Deployment
- 📊 Analytics Dashboard


